# 🚀 README Generator

A simple web app that helps developers generate clean GitHub README files instantly.

## ✨ Features
- Auto README generation
- Markdown formatting
- Copy to clipboard
- Simple and clean UI

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 🚀 How to Use
1. Enter project details
2. Click "Generate README"
3. Copy and use in your project
